# 🗳️ Sistema de Votación Web

Sistema web desarrollado para gestionar procesos de votación electrónica, permitiendo el registro de votos por parte de los usuarios y la administración de resultados mediante un panel de control para administradores.

## 🚀 Características

- Inicio de sesión de usuarios.
- Registro de votos.
- Consulta de votos realizados.
- Administración de usuarios.
- Visualización de resultados de votación.
- Control de acceso según roles.
- Interfaz web intuitiva y fácil de usar.

## 🛠️ Tecnologías Utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap
- JavaScript

## 👥 Roles del Sistema

### Administrador

- Gestionar usuarios.
- Actualizar información de usuarios.
- Consultar votos registrados.
- Visualizar resultados de las votaciones.

### Usuario

- Iniciar sesión.
- Registrar voto.
- Consultar voto realizado.

## 🎯 Objetivo del Proyecto

Desarrollar una plataforma web que permita realizar procesos de votación de manera organizada, segura y eficiente, facilitando el registro de votos y la consulta de resultados en tiempo real.
